### 시기별 Practice 전략

#### **📅 Day 1-2 초반: 처음부터 끝까지 타이핑**

```
목적:
- Python 문법이 완전히 생소한 시기
- 기본 패턴 (변수 선언, print, 연산자) 손에 익히기
- "직접 작성했다"는 자신감 필요
- 타이핑 근육 기억 형성

Practice 작성 방식:
- 스켈레톤 없이 빈 파일에서 시작
- 강사와 함께 한 줄 한 줄 타이핑
- 주석도 함께 작성
```

```python
- 나이 계산하기
```

### 📊 Practice와 Exercise 차별화

| 구분          | Practice             | Exercise             |
| ------------- | -------------------- | -------------------- |
| **목적**      | 강사와 함께 **연습** | 혼자서 **문제 풀이** |
| **난이도**    | 쉬움 → 중간          | 중간 → 어려움        |
| **스켈레톤**  | 많이 제공 (50-80%)   | 적게 제공 (20-30%)   |
| **힌트**      | 상세한 힌트          | 간단한 힌트          |
| **시간**      | 빠르게 (10-20분)     | 충분히 (10-15분)     |
| **강사 역할** | 함께 타이핑/설명     | 질문 대응만          |
| **정답 제공** | 즉시 (같이 작성)     | 나중에 (해설 시간)   |

**핵심 차이:**

- Practice: "**따라하기**" → 빠르게 패턴 익히기
- Exercise: "**직접 해보기**" → 스스로 생각하고 적용

```python
- type() 함수로 타입 확인
- int와 float 타입 이해
- 사칙 연산 및 복합 연산자

# ============================================
# 1. 변수 선언과 할당

# 변수는 데이터를 저장하는 상자입니다

print(count)  # 출력: 15


# ============================================
# 4. 실전 예제: 상품 가격 계산
# ============================================

print(f"정가: {total}원")  # 출력: 정가: 45000원
print(f"할인가: {discounted}원")  # 출력: 할인가: 40500원
```

```python
# 2_practice.py

"""
Day 1 - 2교시: 변수와 숫자형
Practice (강사와 함께 실습)

학습 목표:
- 나이 계산하기
- 온도 변환하기
"""

# ============================================
# 실습 1: 나이 계산하기
# ============================================

# 현재 연도와 태어난 연도를 변수에 저장
current_year = 2026
birth_year = 1995

# 나이 계산
age = current_year - birth_year

# 결과 출력
print("태어난 연도:", birth_year)
print("현재 연도:", current_year)
print("나이:", age, "세")  # 출력: 나이: 31 세


# ============================================
# 실습 2: 온도 변환 (섭씨 → 화씨)
# ============================================

# 섭씨 온도
celsius = 25

# 화씨로 변환
# 공식: 화씨 = (섭씨 × 9/5) + 32
fahrenheit = (celsius * 9/5) + 32

# 결과 출력
print(f"섭씨 {celsius}도는 화씨 {fahrenheit}도입니다")
# 출력: 섭씨 25도는 화씨 77.0도입니다


# ============================================
# 실습 3: BMI 계산기
# ============================================

# 키(cm)와 몸무게(kg)
height_cm = 175
weight = 70

# 키를 미터로 변환 (cm → m)
height_m = height_cm / 100

# BMI 계산
# 공식: 몸무게 / (키 × 키)
bmi = weight / (height_m * height_m)

# 결과 출력
print(f"키: {height_cm}cm, 몸무게: {weight}kg")
print(f"BMI: {bmi:.1f}")  # 출력: BMI: 22.9
```

**총 라인 수**: 약 45줄  
**예상 시간**: 15분 (실습당 5분)  
**방식**: 전체 타이핑

```python
# solutions/solution_shopping.py

"""
Day 1 - 2교시: 변수와 숫자형
Exercise - 쇼핑몰 가격 계산 (정답)
"""

# ============================================
# 주어진 데이터
# ============================================

tshirt_price = 29000
tshirt_quantity = 2
jeans_price = 59000
jeans_quantity = 1
discount_rate = 0.1


# ============================================
# 정답 코드
# ============================================

# 1. 각 상품의 소계
tshirt_subtotal = tshirt_price * tshirt_quantity  # 58000
jeans_subtotal = jeans_price * jeans_quantity  # 59000

# 2. 전체 합계
total = tshirt_subtotal + jeans_subtotal  # 117000

# 3. 할인 금액
discounted_total = total * (1 - discount_rate)  # 105300

# 결과 출력
print(f"티셔츠 소계: {tshirt_subtotal} 원")
print(f"청바지 소계: {jeans_subtotal} 원")
print(f"전체 합계: {total} 원")
print(f"할인 후: {discounted_total} 원")
```
