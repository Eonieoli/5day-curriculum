# PPT 슬라이드 제작 요구사항

## 📋 기본 원칙

```
✅ 한 교시당 PPT 정확히 4장!
✅ 핵심만! 간결하게!
✅ 긴 코드 예제 NO! (주피터에서!)
✅ 강사 설명 스크립트 필수!
```

---

## 📐 슬라이드 구성 (4장 고정)

### 1️⃣ 개념 슬라이드 (1장)
**포함 내용:**
- 정의 (한 문장으로!)
- 사용 이유/목적
- 간단한 비유나 도식
- (선택) FastAPI에서 어떻게 쓰이는지

**작성 원칙:**
- 텍스트 최소화
- 시각적 요소 활용 (📦, ➡️, ✅ 등)
- 학생이 "아하!" 할 수 있게

---

### 2️⃣ 핵심 문법 슬라이드 (1장)
**포함 내용:**
- 기본 문법 구조 (1-3줄 코드만!)
- 가장 많이 쓰는 패턴 1-2개
- 코드 예시는 극도로 간단하게

**작성 원칙:**
- 코드는 5줄 이하!
- 주석으로 핵심만 설명
- "이 정도만 알고 주피터로 가자"

---

### 3️⃣ 주의사항/팁 슬라이드 (1장)
**포함 내용:**
- 🚨 흔한 실수 1-2개
- 💡 꿀팁 1-2개
- (선택) 실무 팁

**작성 원칙:**
- ✅/❌ 로 좋은 예/나쁜 예 비교
- 간결하게!
- 실용적인 것만

---

### 4️⃣ 실습 안내 슬라이드 (1장)
**포함 내용:**
- "이제 Jupyter에서 직접!" 문구
- 📌 실습 목표 3-4개 (체크리스트)
- (선택) 난이도 표시

**작성 원칙:**
- 동기부여!
- 목표 명확하게
- 간결하게

---

## 🎤 강사 설명 스크립트 작성 규칙

### 필수 포함 사항:
```
각 슬라이드마다:

1. 도입 멘트 (자연스럽게)
2. 핵심 설명 (2-3문장)
3. 학생과의 상호작용 (질문 등)
4. 다음 단계 연결
```

### 작성 스타일:
- ✅ 구어체, 자연스럽게
- ✅ 학생에게 직접 말하듯
- ✅ "여러분", "우리" 사용
- ✅ 적절한 강약 조절
- ❌ 딱딱한 문어체
- ❌ 교과서 같은 설명

### 예시:
```
❌ 나쁜 예:
"변수는 값을 저장하는 메모리 공간입니다."

✅ 좋은 예:
"여러분, 변수를 상자라고 생각해보세요. 
숫자나 문자를 담아두는 상자인 거죠. 
나중에 꺼내 쓸 수 있게 이름을 붙여두는 겁니다.
예를 들어 'age'라는 상자에 25를 담아두면,
나중에 age를 부르면 25가 나오는 거예요!"
```

## 📝 요청 시 제공할 정보

새 채팅에서 다음 정보만 주면 됩니다:

```
1. Day X - Y교시 제목
2. 교시 개요 (커리큘럼 복붙)
3. (선택) 특별히 강조할 점
```

예:
```
Day 1 - 2교시: 변수와 숫자형

- 개념: 변수 개념, 메모리 저장, int/float, type()
- Basic: 변수 선언, 사칙 연산, 복합 연산자
- Practice: 나이 계산, 온도 변환
- Exercise: 쇼핑몰 가격 계산, BMI 계산기

특별 강조: type() 함수 중요!
```

## ✅ 체크리스트

제작 완료 후 확인:
```
□ 정확히 4장인가?
□ 긴 코드 예제가 있나? (있으면 삭제!)
□ 강사 설명이 자연스러운가?
□ 실습 목표가 명확한가?
□ 5분 안에 설명 가능한가?
□ 시각적으로 깔끔한가?
```
