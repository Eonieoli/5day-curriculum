### 4교시: 조건문 (if) + Truthy/Falsy ⭐⭐⭐

  - 논리 연산자 복습
  - **Truthy/Falsy** (여기서 처음!):

    ```python
    # False로 취급되는 값
    if "":        # 빈 문자열
    if 0:         # 숫자 0
    if []:        # 빈 리스트
    if {}:        # 빈 딕셔너리
    if None:      # None
    ```
